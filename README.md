
# 📊 Student Performance Analysis & Visualization

A Python-based system for analyzing and visualizing multi-student academic performance.
The project computes statistical measures, detects outliers using the Z-score method, and generates clear visual insights through graphs.

---

## 🚀 Features

* Multi-student input system
* Subject-wise marks entry
* Calculation of mean and standard deviation
* Identification of maximum and minimum marks
* Outlier detection using Z-score
* Clean visualization by excluding outliers
* Bar chart for student performance comparison
* Histogram for marks distribution

---

## 🛠️ Technologies Used

* Python
* NumPy
* Matplotlib

## 📊 Output

* Displays statistical analysis for each student
